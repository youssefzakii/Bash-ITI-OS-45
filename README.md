# Bash-Based Database Management System (DBMS)

This project is a simple **Database Management System (DBMS)** built using **Bash scripting**.  
It provides a command-line interface that allows users to create and manage databases and tables using the Linux file system as persistent storage.

Each database is represented as a directory, while tables are stored as files with accompanying metadata files that define columns and unique keys.

### Database Operations

### Table Operations
- Create tables with column definitions and a unique key
- List tables inside a database
- Drop tables

### Data Operations (CRUD)
- Insert records into tables with unique key validation
- Select records using the unique key
- Update existing records
- Delete records from tables
